- 在前端页面中 不带协议的是相对地址 带协议的是绝对地址  绝对地址=参考地址+相对地址
# 当前端页面写的相对地址前不加斜杠"/"时
- 参考地址默认是当前访问的资源路径  比如localhost:8080/ch06-path/index.jsp的资源路径是localhost:8080/ch06-path/
- 可以加入base标签，指定参考地址恒为某地址
# 当前端页面写的相对地址前加斜杠"/"时
- 参考地址默认是服务器地址，localhost:8080
- 可以在相对地址前加个EL表达式：${pageContext.request.contextPath},代表项目的地址，这样就可以访问到资源了
